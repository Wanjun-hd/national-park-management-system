## 启动步骤

### 后端启动

1. 打开命令行，进入backend目录：
```bash
cd backend
```

2. 激活虚拟环境并启动：
```bash
start.bat
```

后端将运行在：`http://127.0.0.1:8000`

### 前端启动

1. 打开另一个命令行，进入frontend目录：
```bash
cd frontend
```

2. 启动开发服务器：
```bash
npm run dev
```

前端将运行在：`http://localhost:5173`

## 如果仪表盘数据仍显示0

请检查：

1. **后端是否正常运行**
   - 访问：http://127.0.0.1:8000/api/dashboard/stats/
   - 应该看到JSON格式的统计数据

2. **浏览器控制台**
   - 按F12打开开发者工具
   - 查看Console选项卡是否有错误
   - 查看Network选项卡，检查API请求状态

3. **数据库连接**
   - 确认MySQL服务正在运行
   - 确认数据库名为：national_park_system
   - 确认.env文件配置正确
